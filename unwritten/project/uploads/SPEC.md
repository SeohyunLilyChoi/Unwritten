# Unwritten — 기능명세 초안

## 공통 정보

- **서비스명**: Unwritten
- **서비스 성격**: 직장인 암묵지 공유 플랫폼 (직장생활 Q&A + 커뮤니티 + AI 답변)
- **타겟 사용자**: 0~10년차 직장인
- **기술 스택**: React, Tailwind CSS
- **브랜드 톤**: 모던 · 심플 · 스마트. 무겁지 않되 신뢰감 있는 분위기
- **컬러 방향**: 화이트/라이트그레이 베이스, 포인트 컬러 블루(#3B5BFF 계열), 강조 텍스트 블랙
- **타이포**: 한글 위주, 명확한 가독성 중심
- **주요 네비게이션**: 하단 탭바 (홈/질문 · 콘텐츠 · 커뮤니티 · 마이페이지)

---

## 화면 1: 홈 (질문 페이지)

**목적**: AI에게 직장 내 고민/암묵지 관련 질문을 하고 답변을 받는 메인 화면

### UI 구조

- 상단: 앱 로고 (`Unwritten`) 텍스트
- 중간: 질문-답변 피드 (스크롤 가능)
- 하단 고정: 질문 입력창

### 질문 입력창

- Placeholder: `오늘의 고민은 무엇인가요?`
- 멀티라인 텍스트 입력 (textarea)
- 우측 하단: 전송 버튼 (아이콘 버튼, 텍스트 없을 때 비활성)
- 키보드 올라올 때 입력창이 키보드 위에 고정 (iOS safe area 대응)

**상태**
- 기본: 회색 placeholder
- 입력 중: 텍스트 표시, 전송 버튼 활성화
- 전송 후: 로딩 → 답변 렌더링

### 질문-답변 카드 (피드 아이템)

**질문 영역**
- 접기/펼치기 토글 (chevron 아이콘)
- 질문 텍스트 (굵은 제목 형태)
- 접혔을 때: 제목만 표시
- 펼쳤을 때: 질문 본문 말풍선 형태로 노출 (회색 배경 카드)

**AI 답변 영역**
- `AI 답변` 레이블 (파란 점 + 텍스트)
- 답변 요약 헤드라인 (굵고 크게)
- 핵심 문장 강조 (파란색 밑줄 텍스트)
- 분석 근거 텍스트 (예: `콘텐츠 8건 · 커뮤니티 124건을 분석했어요.`)

**데이터 카드** (답변 하단에 포함)
- 제목 + 총 응답자 수
- 막대 바 차트 형태의 통계 (선택지별 퍼센트, 최다 항목 강조)
- `관련 데이터 더보기 >` 링크 버튼

**관련 커뮤니티 글 카드**
- 섹션 제목 + `전체 보기` 링크
- 커뮤니티 글 2~3개 미리보기
  - 글 제목 (굵게)
  - 본문 2줄 미리보기
  - 댓글 수 / 좋아요 수

**추가 액션 버튼** (답변 확인 후 노출)
- `+` 버튼 → `추가 질문하기` / `피드백하기` 버튼으로 전환
- `추가 질문하기`: 같은 주제로 추가 질문 입력창 포커스
- `피드백하기`: 답변 피드백 모달 오픈
- `X` 버튼: 액션 버튼 닫기

**상태**
- 질문 접힘 / 질문 펼침
- 답변 로딩 중 (skeleton UI)
- 답변 완료 + 액션 버튼 미노출
- 답변 완료 + 액션 버튼 노출 (+ 탭 후)
- 빈 상태 (첫 질문 전): 안내 텍스트 또는 예시 질문 표시

## 화면 2: 콘텐츠 페이지

**목적**: 직장 내 암묵지 관련 콘텐츠를 개인화된 피드 형태로 제공

### UI 구조

- 상단: 페이지 타이틀 + 필터/탭
- 콘텐츠 피드 (스크롤)

### 콘텐츠 유형

**① 투표 / 밸런스게임 카드**
- 질문 텍스트
- 선택지 2개 버튼
- 선택 전: 버튼 기본 상태
- 선택 후: 결과 비율 표시 (막대 or 퍼센트)
- 총 참여자 수 표시

**② 아티클 카드**
- 썸네일 이미지
- 카테고리 태그
- 제목 / 요약 2줄
- 읽기 시간 or 날짜

**③ 사회생활 콘테스트 카드**
- 참여 유도 CTA 포함

### 개인화 피드 로직 (UI에 반영할 것)
- 최근 질문 키워드 기반 추천 태그 표시
- `나를 위한 추천` 섹션 상단 고정

**상태**
- 로딩: 스켈레톤 카드
- 투표 전 / 투표 후
- 스크랩 전 / 스크랩 후 (북마크 아이콘)
- 빈 피드: `아직 추천 콘텐츠가 없어요` 안내

## 화면 3: 커뮤니티 페이지

**목적**: 사용자들이 직장생활 질문/고민을 올리고 소통하는 공간

### UI 구조

- 상단: 검색바 + 정렬 필터 (최신순 / 인기순)
- 내 연차/직무 기반 트렌드 요약 카드 (상단 고정)
- 게시글 리스트
- 우측 하단 FAB: 글쓰기 버튼

### 트렌드 요약 카드

- `나와 비슷한 {직무} {연차}의 관심사` 타이틀
- 키워드 태그 또는 간단한 시각화

### 게시글 카드

- 제목
- 본문 2줄 미리보기
- 작성자 정보 (익명 or 직무/연차만 표시)
- 댓글 수 / 좋아요 수
- AI 요약 묶음 뱃지 (유사 질문이 묶인 경우 표시)

### AI 묶음 질문 뷰

- 유사 질문 N개 묶음 표시
- 답변 요약 제공
- 개별 원문 글 펼쳐보기 가능

### 게시글 상세

- 본문 전체
- 댓글 목록 (채택된 댓글 상단 고정 + 뱃지)
- 댓글 입력창 (하단 고정)
- 좋아요 / 스크랩 / 공유 액션

**상태**
- 게시글 리스트 로딩
- 빈 리스트: `아직 게시글이 없어요`
- 글쓰기 모달/화면: 제목 + 본문 + 카테고리 선택
- 채택 전 / 채택 후 댓글

## 화면 4: 마이페이지

**목적**: 내 활동 내역 확인 및 계정 관리

### UI 구조

- 상단: 프로필 영역 (직무/연차 표시, 수정 버튼)
- 활동 요약 카드 (포인트, 채택 수, 스크랩 수)
- 메뉴 리스트

### 프로필 영역

- 직무 / 연차 텍스트
- 편집 버튼 → 프로필 수정 화면 이동

### 활동 요약

- 보유 포인트 (채택 답변으로 적립)
- 내가 쓴 글 수 / 채택된 답변 수

### 메뉴 항목

- 내가 쓴 글
- 스크랩한 콘텐츠
- 포인트 내역
- 알림 설정
- 앱 정보 / 로그아웃

### 사용자 분석 카드 (선택적 노출)

- 활동 기반 인사이트 (예: `당신은 주로 조직문화 관련 질문을 많이 해요`)

**상태**
- 로그인 상태 / 비로그인 상태 (로그인 유도 CTA)
- 포인트 0 / 포인트 있음

| 컴포넌트 | 설명 |
|---------|------|
| 하단 탭바 | 홈(질문) / 콘텐츠 / 커뮤니티 / 마이페이지, 활성 탭 파란색 표시 |
